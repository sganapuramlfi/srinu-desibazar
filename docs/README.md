# DesiBazaar Technical Documentation

## Overview
DesiBazaar is a multi-industry business platform providing scalable technological solutions for entrepreneurs across Salon, Restaurant, Event, Real Estate, and Retail sectors. This documentation covers the technical implementation, architecture decisions, and development guidelines.

## Quick Start
- Frontend: React.js with TypeScript
- Backend: Express.js
- Database: PostgreSQL with Drizzle ORM
- Authentication: Passport.js with session management
- UI Components: shadcn/ui with Tailwind CSS
